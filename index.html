About this project
(What I learned):

> HTML

  • HTML Structure: Understanding the basic structure of an HTML document with <!DOCTYPE html>, <html>, <head>, and <body> tags.
  • Meta Tags: Utilizing <meta charset="UTF-8"> for character encoding and <meta name="viewport" content="width=device-width, initial-scale=1.0"> for responsive design.
  • Title Tag: Setting the document title with <title> tag.
  • External Resources: Linking to external resources with <link> tags for fonts and CSS (style.css).
  • Script Tag: Loading JavaScript asynchronously using <script src="script.js" defer></script>.
  • Semantic Tags: Using semantic tags like <header>, <main>, <footer>, and <form> for better structure and accessibility.
  • Image Tag: Inserting images with <img> and providing alternative text with alt attribute for accessibility.
  • Form Handling: Creating a form (<form>) to accept user input (<input> and <button>) with basic validation (required attribute).
  • Audio Tag: Embedding audio files (<audio>) for background music (loop attribute) and sound effects.
  • Button Elements: Creating interactive buttons (<button>) for user interaction.
  • CSS Styling: Referencing an external CSS file (style.css) for styling the HTML elements.

> CSS

  • @font-face Rule: Including custom fonts using @font-face. Specifying font format (format()) and URL (src) for the font file.
  • CSS Variables: Defining variables within :root selector (--variable-name). Creating themes with color variables.
  • Responsive Design: Using min() function for responsive font sizes. Setting maximum widths with max-width for responsive layouts.
  • Box Model: Understanding and applying box-sizing, padding, margin, and border properties for layout control.
  • Flexbox Layout: Utilizing display: flex for flexible layout alignment. Implementing align-items, justify-content, and flex-direction properties for layout flexibility.
  • Grid Layout: Creating grid layouts with display: grid. Defining grid templates with grid-template-rows and grid-template-columns for structured layouts.
  • Animations: Defining animations using @keyframes. Applying animations with animation property. Understanding animation timing functions (ease, linear, etc.) and delays.
  • Transitions: Using transition to smoothly animate property changes.
  • Pseudo-elements: Creating and styling ::before pseudo-elements for additional content.
  • Media Queries: Utilizing viewport units (vw, vh) for responsive design adjustments.
  • Layering and Positioning: Managing stacking order with z-index. Positioning elements absolutely using position: absolute.
  • Text Styling: Applying different fonts with font-family. Styling text using font-weight, font-size, text-align, and text-shadow.
  • Button Styling: Customizing buttons with background-color, border, and border-radius. Adding interactive states with :hover and :active pseudo-classes.
  • Backgrounds: Applying background colors and gradients. Using background-size and background-position for background styling.
  • Masks and Clipping: Creating masked elements with -webkit-mask and mask properties.
  • Opacity and Visibility: Controlling element visibility using opacity. Managing element display with display property.
  • CSS Selectors: Utilizing various selectors including class (.), ID (#), and element selectors. Combining selectors for specific targeting (e.g., .battle::before).

> Javascript

  • DOM Manipulation: Interacting with HTML elements using various methods like querySelector, getElementById, createElement, and manipulating their attributes and content.
  • Event Handling: Utilizing event listeners (addEventListener) to respond to user interactions such as clicks and form submissions.
  • Conditional Logic: Using if, else if, and else statements to control program flow based on different conditions (e.g., game outcomes).
  • Functions: Defining and calling functions (playRound, displayText, renderBar, etc.) to encapsulate and reuse blocks of code.
  • Animation: Implementing CSS animations (animationend event) to enhance visual feedback during gameplay.
  • Arrays and Loops: Working with arrays (querySelectorAll, forEach) and loops (for, forEach) to iterate through and manipulate collections of elements.
  • Audio Control: Managing audio playback (play, pause) and setting audio attributes (currentTime) dynamically based on game state and user preferences.
  • CSS Manipulation: Dynamically adding and removing CSS classes (classList) to modify element styles and trigger animations.
  • Form Handling: Handling form submissions (addEventListener for submit) and input changes (addEventListener for input) to update game settings dynamically.
  • Viewport Detection: Checking if elements are within the viewport (getBoundingClientRect, isInViewport) to trigger animations and behaviors.
